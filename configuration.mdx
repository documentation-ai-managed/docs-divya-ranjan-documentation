### Database Setup

Connect Pattrns to your existing databases for seamless data import:

<CodeGroup tabs="PostgreSQL,MySQL,MongoDB">
  ```sql
-- PostgreSQL connection example
CREATE EXTENSION IF NOT EXISTS "uuid-ossp";

-- Configure Pattrns schema
CREATE SCHEMA pattrns;
CREATE TABLE pattrns.data_sources (
  id UUID PRIMARY KEY DEFAULT uuid_generate_v4(),
  name VARCHAR(255) NOT NULL,
  type VARCHAR(50) NOT NULL,
  config JSONB,
  created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);
  ```
  ```sql
-- MySQL connection example
CREATE DATABASE pattrns_db;
USE pattrns_db;

CREATE TABLE data_sources (
  id INT AUTO_INCREMENT PRIMARY KEY,
  name VARCHAR(255) NOT NULL,
  type VARCHAR(50) NOT NULL,
  config JSON,
  created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);
  ```

// MongoDB connection example

### API Integration Setup

Configure Pattrns to pull data from external APIs:

```python
import requests

class PattrnsAPIClient:
    def __init__(self, base_url, api_key):
        self.base_url = base_url
        self.api_key = api_key
        self.session = requests.Session()
        self.session.headers.update({
            'Authorization': f'Bearer {api_key}',
            'Content-Type': 'application/json'
        })
    
    def configure_endpoint(self, endpoint_config):
        """Configure a new API endpoint for data import"""
        config = {
            'name': endpoint_config['name'],
            'url': endpoint_config['url'],
            'method': endpoint_config.get('method', 'GET'),
            'headers': endpoint_config.get('headers', {}),
            'auth': endpoint_config.get('auth'),
            'rate_limit': endpoint_config.get('rate_limit', 100)
        }
        
        response = self.session.post(
            f'{self.base_url}/api/config/endpoints',
            json=config
        )
        return response.json()

# Example usage
client = PattrnsAPIClient('https://api.pattrns.com', 'your-api-key')
endpoint = {
    'name': 'Weather API',
    'url': 'https://api.weather.com/v1/data',
    'method': 'GET',
    'headers': {'Accept': 'application/json'},
    'rate_limit': 50
}
result = client.configure_endpoint(endpoint)
print(f"Configured endpoint: {result['id']}")
```

## Configuration File Example

Here's a complete configuration file example for a typical Pattrns deployment:

```yaml
# Pattrns Configuration File
version: '2.0'

# Basic settings
app:
  name: 'My Pattrns Instance'
  port: 8080
  environment: 'production'

# Database configuration
database:
  type: 'postgresql'
  host: 'localhost'
  port: 5432
  database: 'pattrns_prod'
  username: 'pattrns_user'
  password: '${DB_PASSWORD}'
  ssl: true

# Authentication
auth:
  provider: 'oauth2'
  client_id: '${OAUTH_CLIENT_ID}'
  client_secret: '${OAUTH_CLIENT_SECRET}'
  redirect_uri: 'https://myapp.pattrns.com/auth/callback'

# Data processing
processing:
  max_workers: 4
  memory_limit: '2GB'
  timeout: 300
  cache_enabled: true

# Integrations
integrations:
  slack:
    enabled: true
    webhook_url: '${SLACK_WEBHOOK}'
  email:
    smtp_host: 'smtp.gmail.com'
    smtp_port: 587
    username: '${EMAIL_USER}'
    password: '${EMAIL_PASS}'
```

<Callout kind="tip">Use environment variables for sensitive information like passwords and API keys to keep them secure.</Callout>
